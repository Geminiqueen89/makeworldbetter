# Knowledge is Power

A personal, educational static website dedicated to mental health awareness, recovery, healthy relationships, self-care, and personal growth.

## Purpose
This site provides reliable, empowering information on topics like:
- Narcissism and abuse recovery
- Domestic violence and types of abuse
- Mental health conditions (ADHD, bipolar, depression, anxiety)
- Stress management, sleep hygiene, CBT techniques
- Addictions (substance, gambling)
- Dual diagnosis and trauma-informed care
- Healthy vs. toxic relationships, boundaries, self-esteem building
- Resources, affirmations, quotes, and practical toolkits

**Important Disclaimer**: All content is for educational purposes only. It is not medical, psychological, or therapeutic advice. Always consult a licensed professional for personal concerns. In crisis? Call or text 988 (US Suicide & Crisis Lifeline).

## Technologies Used
- HTML5
- CSS3 (mobile-first, responsive design with hamburger menu)
- Vanilla JavaScript (for interactive checklist in Self-Care Toolkit and basic quizzes)
- No frameworks, backend, or databases — fully static site

## How to View Locally
1. Open the folder in your file explorer.
2. Double-click `index.html` — it should open in your default browser.
3. Resize the window or use browser dev tools (F12 → device toolbar) to test mobile view.

## How to Update
- Edit any .html file directly (use a text editor like Notepad++, VS Code, or even Google Docs → export as plain text).
- Update `styles.css` for design changes.
- Add new pages? Create a new .html file and link it in the `<nav>` section of every page.

## Deployment (Hosting Options)
### Netlify (Recommended – Free & Easy)
1. Go to https://app.netlify.com
2. Sign up/login (free with email/Google/GitHub)
3. Drag and drop your entire folder onto the Netlify dashboard
4. Wait ~30 seconds → get a live URL (e.g., https://knowledge-is-power-123.netlify.app)

### GitHub Pages (Free)
1. Create a free GitHub account
2. New repository → upload all files
3. Settings → Pages → select main branch → root folder → Save
4. Site live at https://yourusername.github.io/repo-name

### Vercel (Alternative)
Similar drag-and-drop or Git deploy: https://vercel.com
