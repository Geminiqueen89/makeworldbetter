# Knowledge is Power

A personal, educational static website dedicated to mental health awareness, recovery, healthy relationships, self-care, and personal growth.

## Purpose
This site provides reliable, empowering information on topics like:
- Narcissism and abuse recovery
- Domestic violence and types of abuse
- Mental health conditions (ADHD, bipolar, depression, anxiety)
- Stress management, sleep hygiene, CBT techniques
- Addictions (substance, gambling)
- Dual diagnosis and trauma-informed care
- Healthy vs. toxic relationships, boundaries, self-esteem building
- Resources, affirmations, quotes, and practical toolkits

**Important Disclaimer**: All content is for educational purposes only. It is not medical, psychological, or therapeutic advice. Always consult a licensed professional for personal concerns. In crisis? Call or text 988 (US Suicide & Crisis Lifeline).

## Technologies Used
- HTML5
- CSS3 (mobile-first, responsive design with hamburger menu)
- Vanilla JavaScript (for interactive checklist in Self-Care Toolkit and basic quizzes)
- No frameworks, backend, or databases — fully static site
